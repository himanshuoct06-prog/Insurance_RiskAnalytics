# Insurance Risk Analytics Project

## Project Overview
This project focuses on Insurance Risk Analytics using Data Engineering, Data Analysis, Machine Learning, and Dashboarding techniques.

The solution includes:
- Data Cleaning and Transformation
- Exploratory Data Analysis (EDA)
- Power BI Dashboards
- Machine Learning Risk Prediction
- Streamlit Web Application Deployment

---

## Technologies Used
- Python
- SQL
- Snowflake
- Databricks
- Power BI
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib

---

## Key Features
- Customer Risk Prediction
- Claims Analysis Dashboard
- Insurance Operations Dashboard
- KPI Monitoring
- Real-time Risk Classification
- Interactive Visualizations

---

## Machine Learning Model
- Random Forest Classifier
- Accuracy: 83%
- ROC-AUC Score: 0.90

---

## Deployment
The application is deployed using:
- GitHub
- Streamlit Cloud

## Project Outcome
The project helps insurance companies:
- Identify high-risk customers
- Improve claim monitoring
- Enable data-driven decision making
- Optimize operational efficiency
